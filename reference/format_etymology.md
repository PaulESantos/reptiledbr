# Format etymological data of reptiles **\[experimental\]** Extracts and formats the etymological information associated with reptile species.

Format etymological data of reptiles **\[experimental\]** Extracts and
formats the etymological information associated with reptile species.

## Usage

``` r
format_etymology(reptile_data)
```

## Arguments

- reptile_data:

  A data frame obtained using the `get_reptile_species_data()` function.

## Value

A tibble containing the formatted etymological details for each species.
