# reptiledbr: Interface to the Reptile Database for Querying and Retrieving Taxonomic Data

Provides tools to search, access, and format taxonomic information from
the Reptile Database (<http://reptile-database.org>) directly within R.
Users can retrieve species-level data, distribution, etymology,
synonyms, common names, and other relevant information for reptiles.
Designed for taxonomists, ecologists, and biodiversity researchers.
