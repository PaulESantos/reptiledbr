Format bibliographic reference data of reptiles **\[experimental\]**
Extracts and formats bibliographic references associated with reptile
species.

## Usage

``` r
format_references(reptile_data)
```

## Arguments

- reptile_data:

  A data frame obtained using the `get_reptile_species_data()` function.

## Value

A tibble containing formatted references for each species.
